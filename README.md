# Projeto ISPSoft – Roscas de Elevação

Este projeto controla as roscas de elevação de couro de um picador industrial. 
Inclui:
- Controle remoto independente das roscas;
- Intertravamento para evitar reversão durante a partida;
- Botão liga/desliga e emergência.

## Tecnologias
- ISPSoft para programação;
- Sistemas elétricos e eletrônicos industriais.
